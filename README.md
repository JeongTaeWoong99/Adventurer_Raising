## 프로젝트 소개
판타지 배경의 쿼터뷰 3D 멀티 RPG 게임입니다. 

데디케이트 서버를 구축하여, 멀티플레이가 가능합니다.

3D 게임 개발 및 네트워크 시스템에 대한 전반적인 이해를 목적으로, 클라이언트부터 서버와 DB까지 직접 설계하고 구현했습니다.
